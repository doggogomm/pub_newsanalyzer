# 뉴스 분석 애플리케이션 사용 설명서

이 애플리케이션은 CSV 또는 Excel 파일에 저장된 뉴스 기사 데이터를 분석하고 시각화할 수 있는 GUI 프로그램입니다.

## 설치 방법

1. 필요한 패키지 설치:

```bash
pip install -r requirements.txt
```

필수 패키지:
- pandas
- numpy
- matplotlib

선택 패키지 (추가 기능 사용 시 필요):
- wordcloud (워드클라우드 생성 기능)
- konlpy (한국어 자연어 처리 기능)
- openpyxl (Excel 파일 읽기)

## 사용 방법

1. 프로그램 실행:

```bash
python news_analysis_app.py
```

2. 기본 사용 단계:
   - 파일 로드: CSV 또는 Excel 파일 선택
   - 컬럼 선택: 날짜, 제목, 내용, 매체명 등의 컬럼 지정
   - 기간 설정: 분석할 시간 범위 선택
   - 불용어 설정: 제외할 단어 목록 작성
   - 분석 실행: 선택한 설정으로 데이터 분석 시작

## 주요 기능

### 1. 파일 로드
- CSV (.csv) 및 Excel (.xlsx, .xls) 파일 지원
- 자동 인코딩 감지 (기본값: UTF-8)

### 2. 컬럼 선택
- 날짜/시간 컬럼: 시간에 따른 분석용
- 제목 컬럼: 기사 제목 (필수)
- 내용 컬럼: 본문 텍스트 (워드클라우드 생성 시 필요)
- 매체명 컬럼: 매체별 분석용

### 3. 기간 설정
- 기간 유형: 일별, 월별, 년별 분석 가능
- 시작일/종료일: 특정 기간 내 데이터만 분석

### 4. 불용어 설정
- 사용자 지정 불용어 추가 가능
- 기본 불용어 목록 제공
- 쉼표(,)나 공백으로 구분하여 여러 단어 입력 가능

### 5. 시각화
- 기간별 분석: 시간에 따른 기사 수 변화 그래프
- 매체별 분석: 매체별 기사 수 막대 그래프
- 워드클라우드: 기사 내용에서 추출한 주요 단어 시각화 (wordcloud 패키지 필요)

## 한국어 처리

- konlpy 패키지가 설치되어 있으면 한국어 형태소 분석을 통해 명사만 추출
- 패키지가 없는 경우 기본 토큰화 방식으로 대체
- 한글 폰트가 시스템에 설치되어 있어야 워드클라우드에서 한글이 제대로 표시됨

## 주의사항

1. 대용량 파일을 처리할 때는 시간이 오래 걸릴 수 있습니다.
2. 날짜 형식이 지정한 형식과 일치해야 날짜 필터링이 제대로 작동합니다.
3. wordcloud와 konlpy는 선택적 패키지로, 설치되지 않은 경우 관련 기능이 제한됩니다.

## 문제 해결

- 한글 깨짐 문제가 발생할 경우, 시스템에 한글 폰트가 설치되어 있는지 확인하세요.
- 날짜 변환 오류가 발생할 경우, 날짜 형식을 실제 데이터와 일치하도록 수정하세요.
- 메모리 오류가 발생하면 더 작은 데이터셋으로 나누어 분석하세요.
